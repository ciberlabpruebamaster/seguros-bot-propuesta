📄 Workflow JSON — Pegad en n8n
json — n8n workflow 
// Pegad este JSON en n8n: Workflows → Import from clipboard
{
  "name": "Unicom Seguros — WhatsApp Bot",
  "nodes": [
    {
      "name": "WhatsApp Trigger",
      "type": "n8n-nodes-base.whatsAppTrigger",
      "parameters": { "events": ["messages"] }
    },
    {
      "name": "Parse Message",
      "type": "n8n-nodes-base.code",
      "parameters": {
        "code": "const msg = $input.first().json.entry[0].changes[0].value.messages[0];\nconst contact = $input.first().json.entry[0].changes[0].value.contacts[0];\nreturn {\n  from: msg.from,\n  text: msg.text?.body?.toLowerCase().trim() || '',\n  type: msg.type,\n  name: contact.profile.name,\n  timestamp: msg.timestamp\n};"
      }
    },
    {
      "name": "Router",
      "type": "n8n-nodes-base.switch",
      "parameters": {
        "rules": {
          "values": [
            { "conditions": [{ "value1": "={{ $json.text }}", "operation": "contains", "value2": "1" }], "output": 0 },
            { "conditions": [{ "value1": "={{ $json.text }}", "operation": "contains", "value2": "2" }], "output": 1 },
            { "conditions": [{ "value1": "={{ $json.text }}", "operation": "contains", "value2": "3" }], "output": 2 },
            { "conditions": [{ "value1": "={{ $json.text }}", "operation": "contains", "value2": "4" }], "output": 3 },
            { "conditions": [{ "value1": "={{ $json.text }}", "operation": "contains", "value2": "6" }], "output": 4 }
          ]
        }
      }
    },
    {
      "name": "Check Business Hours",
      "type": "n8n-nodes-base.code",
      "parameters": {
        "code": "const now = new Date();\nconst hour = now.getHours();\nconst day = now.getDay(); // 0=Dom, 6=Sab\nconst isBusinessHours = day >= 1 && day <= 5 && hour >= 9 && hour < 18;\nreturn { ...($input.first().json), isBusinessHours };"
      }
    },
    {
      "name": "Send WhatsApp",
      "type": "n8n-nodes-base.whatsApp",
      "parameters": {
        "operation": "sendMessage",
        "phoneNumberId": "={{ $env.WA_PHONE_NUMBER_ID }}",
        "recipientPhoneNumber": "={{ $json.from }}",
        "textBody": "={{ $json.response }}"
      }
    }
  ]
}

🚀 Guía de implementación paso a paso
1
Crear cuenta Meta Business
Acceder a business.facebook.com y crear o verificar vuestra cuenta de empresa. Necesitaréis documentación oficial de la empresa para la verificación.
💡 La verificación puede tardar 2-5 días hábiles. Iniciadla cuanto antes.
2
Crear app en Meta Developer Portal
En developers.facebook.com: Crear nueva app → tipo "Business" → añadir producto "WhatsApp". Aquí obtendréis el Phone Number ID y el Access Token.
3
Instalar y configurar n8n
Opción recomendada: n8n Cloud (sin servidor). Alternativa: Docker en vuestro servidor.

docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n
💡 n8n Cloud tiene plan gratuito suficiente para empezar.
4
Configurar webhook de WhatsApp
En la app de Meta, sección "Webhooks", añadid la URL del trigger de n8n:
https://vuestro-n8n.com/webhook/whatsapp-unicom

Subscribid a los eventos: messages, message_deliveries, message_reads
5
Importar workflow en n8n
En n8n: Workflows → New → Import from clipboard → pegar el JSON de la pestaña "Flujo n8n". Configurad las credenciales de WhatsApp Business API.
6
Enviar plantillas a aprobación
En Meta Business Manager → WhatsApp → Plantillas de mensajes → Nueva plantilla. Añadid las 6 plantillas de la pestaña "Plantillas Meta". Tiempo de aprobación: 24-48h.
⚠️ Sin plantillas aprobadas, el bot solo puede responder a mensajes que el usuario inicia primero (ventana de 24h).
7
Pruebas y activación
Realizad pruebas con vuestro propio número primero. Verificad cada flujo: cotización, siniestro, pagos, derivación a agente dentro y fuera de horario. Cuando todo funcione, activad el número de WhatsApp oficial de Unicom Seguros.
✅ ¡Listo! El bot empezará a atender a vuestros clientes automáticamente.
